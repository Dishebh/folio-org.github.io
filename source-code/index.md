## Server-side Toolkit

Okapi API Gateway proxy/discovery/deployment service.

[okapi-infrastructure](https://github.com/sling-incubator/okapi-infrastructure) -
Okapi microservices infrastructure.

[auth_module_prototype](https://github.com/sling-incubator/auth_module_prototype) -
Prototype of a JWT auth module for Okapi.

[lsp-apis-impl](https://github.com/sling-incubator/lsp-apis-impl) -
Implementation options for REST API.
